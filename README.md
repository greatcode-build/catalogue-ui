# Catalogue UI

A lightweight product showcase UI built with HTML, CSS, and JavaScript.

## What this project does

`Catalogue UI` renders a modern product card slider using the Swiper library. It presents a set of product images with titles and prices in an interactive card effect.

## Why it’s useful

- Demonstrates a polished product carousel layout
- Uses minimal frontend code with no build step
- Shows how to integrate `Swiper` cards effect via CDN
- Good starting point for e-commerce and catalog landing pages

## Key features

- Responsive card-style slider
- Smooth swipe and drag interactions
- Product image overlay with name and price
- Simple file-based asset structure

## Getting started

### Prerequisites

- A modern web browser

### Run locally

1. Clone the repository:

```bash
git clone <repository-url>
cd catalogue-ui
```

2. Open `index.html` in your browser, or serve the folder with a local server.

### Example using Python

```bash
cd catalogue-ui
```

Then open `http://localhost:8000`.

## Project structure

- `index.html` — main page markup
- `css/style.css` — styling for the slider and layout
- `js/main.js` — Swiper initialization
- `images/` — product image assets

## Customization

- Update slide content in `index.html` under `.swiper-wrapper`
- Change styling in `css/style.css`
- Adjust slider behavior in `js/main.js`
